2. **metadata.ts** -- Create new file following the read.ts module augmentation pattern:

   **Imports:** `CString, ptr, read` from `bun:ffi`, `Database` from `./database`, `check` from `./errors`, `allocPointerOut, readPointerOut, toCString` from `./ffi-helpers`, `getSymbols` from `./loader`.

   **Types** (exported):
   ```typescript
   export interface ScalarMetadata {
     name: string;
     dataType: number;
     notNull: boolean;
     primaryKey: boolean;
     defaultValue: string | null;
     isForeignKey: boolean;
     referencesCollection: string | null;
     referencesColumn: string | null;
   }

   export interface GroupMetadata {
     groupName: string;
     dimensionColumn: string | null;
     valueColumns: ScalarMetadata[];
   }
   ```

   **Constants:** `SCALAR_METADATA_SIZE = 56`, `GROUP_METADATA_SIZE = 32`.

   **Helper: `readNullableString(base, offset)`** -- reads `read.ptr(base, offset)`, returns `null` if pointer is 0/falsy, otherwise `new CString(ptr).toString()`.

   **Helper: `readScalarMetadata(base, byteOffset)`** -- reads all 8 fields from the struct at the given offset using the verified layout. Uses `read.ptr()` for strings (with null check via readNullableString for default_value, references_collection, references_column), `read.i32()` for int fields, converts ints to booleans with `!== 0`.

   **Helper: `readGroupMetadata(base, byteOffset)`** -- reads group_name (ptr at +0), dimension_column (nullable ptr at +8), value_columns pointer (ptr at +16), value_column_count (`Number(read.u64(base, byteOffset + 24))`). Iterates value_columns pointer with stride 56 calling `readScalarMetadata`.

   **Module augmentation** (`declare module "./database"`): Add 8 methods to the Database interface:
   - `getScalarMetadata(collection: string, attribute: string): ScalarMetadata`
   - `getVectorMetadata(collection: string, groupName: string): GroupMetadata`
   - `getSetMetadata(collection: string, groupName: string): GroupMetadata`
   - `getTimeSeriesMetadata(collection: string, groupName: string): GroupMetadata`
   - `listScalarAttributes(collection: string): ScalarMetadata[]`
   - `listVectorGroups(collection: string): GroupMetadata[]`
   - `listSetGroups(collection: string): GroupMetadata[]`
   - `listTimeSeriesGroups(collection: string): GroupMetadata[]`

   **Get method implementations** (4 methods via `Database.prototype.X = function`):
   - Allocate a `Uint8Array` of struct size (56 for scalar, 32 for group)
   - Call the C API function with `handle, toCString(collection), toCString(name), ptr(outBuf)`
   - Read the struct into a JS object via the helper
   - Call the corresponding free function: `quiver_database_free_scalar_metadata(ptr(outBuf))` or `quiver_database_free_group_metadata(ptr(outBuf))`
   - Return the JS object

   **List method implementations** (4 methods via `Database.prototype.X = function`):
   - Allocate `allocPointerOut()` for the array pointer and `new BigUint64Array(1)` for the count
   - Call the C API function with `handle, toCString(collection), ptr(outArray), ptr(outCount)`
   - Read count via `Number(outCount[0])`. If 0, return `[]`.
   - Read the array pointer via `readPointerOut(outArray)`
   - Iterate with stride (56 for scalar, 32 for group), building result array
   - Call `quiver_database_free_scalar_metadata_array(arrPtr, count)` or `quiver_database_free_group_metadata_array(arrPtr, count)`
   - Return the result array
