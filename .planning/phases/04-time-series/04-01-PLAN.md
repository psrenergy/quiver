Define data type constants: `DATA_TYPE_INTEGER = 0`, `DATA_TYPE_FLOAT = 1`, `DATA_TYPE_STRING = 2`, `DATA_TYPE_DATE_TIME = 3`.

**readTimeSeriesGroup implementation:**
- Allocate 5 out-params: `outColumnNames` (allocPointerOut), `outColumnTypes` (allocPointerOut), `outColumnData` (allocPointerOut), `outColumnCount` (BigUint64Array(1)), `outRowCount` (BigUint64Array(1)).
- Call `lib.quiver_database_read_time_series_group(handle, toCString(collection), toCString(group), id, ptr(outColumnNames), ptr(outColumnTypes), ptr(outColumnData), ptr(outColumnCount), ptr(outRowCount))` via check().
- Read colCount and rowCount from BigUint64Array as Number.
- If colCount === 0, return empty object `{}`.
- Dereference pointers: `namesPtr = readPointerOut(outColumnNames)`, `typesPtr = readPointerOut(outColumnTypes)`, `dataPtr = readPointerOut(outColumnData)`.
- Read column types as Int32Array via `toArrayBuffer(typesPtr, 0, colCount * 4)`.
- For each column c (0 to colCount-1):
  - Read column name: `readPtrAt(namesPtr, c * 8)` -> `new CString(namePtr).toString()`.
  - Read column data pointer: `readPtrAt(dataPtr, c * 8)`.
  - Dispatch on type: INTEGER -> `toArrayBuffer(colDataPtr, 0, rowCount * 8)`, `new BigInt64Array(buf)`, convert to number[]. FLOAT -> `toArrayBuffer(colDataPtr, 0, rowCount * 8)`, `new Float64Array(buf)`, `Array.from()`. STRING/DATE_TIME -> iterate rows, `readPtrAt(colDataPtr, r * 8)`, `new CString(strPtr).toString()`.
- Call `lib.quiver_database_free_time_series_data(namesPtr, typesPtr, dataPtr, colCount, rowCount)` to free C memory.
- Return result object.

**updateTimeSeriesGroup implementation:**
- Get entries from data: `Object.entries(data)`. If empty (length === 0), pass null/0/0 to clear: `check(lib.quiver_database_update_time_series_group(handle, toCString(collection), toCString(group), id, null, null, null, 0, 0))` and return.
- Determine columnCount and rowCount from entries.
- Build 3 parallel native arrays + keepalive:
  - `namesPtrTable` (BigInt64Array of column name CString pointers)
  - `typesArr` (Int32Array of column types, use Number.isInteger on first element to distinguish INTEGER vs FLOAT; all strings are DATE_TIME=3 since time series dimension is always string/datetime type -- actually use DATA_TYPE_STRING=2 since C API treats STRING and DATE_TIME as interchangeable)
  - `dataPtrTable` (BigInt64Array of pointers to typed data arrays)
- For each column: allocate typed array (BigInt64Array for integers, Float64Array for floats, BigInt64Array pointer table for strings), push to keepalive.
- Call `lib.quiver_database_update_time_series_group(handle, toCString(collection), toCString(group), id, ptr(namesPtrTable), ptr(typesArr), ptr(dataPtrTable), columnCount, rowCount)` via check().

**hasTimeSeriesFiles implementation:**
- `outResult = new Int32Array(1)`, call `lib.quiver_database_has_time_series_files(handle, toCString(collection), ptr(outResult))` via check(), return `outResult[0] !== 0`.

**listTimeSeriesFilesColumns implementation:**
- `outColumns = allocPointerOut()`, `outCount = new BigUint64Array(1)`.
- Call `lib.quiver_database_list_time_series_files_columns(handle, toCString(collection), ptr(outColumns), ptr(outCount))`.
- Read count; if 0 return []. Dereference outColumns, iterate with `readPtrAt(columnsPtr, i * 8)` -> CString.
- Free with `lib.quiver_database_free_string_array(columnsPtr, count)`.

**readTimeSeriesFiles implementation:**
- `outColumns = allocPointerOut()`, `outPaths = allocPointerOut()`, `outCount = new BigUint64Array(1)`.
- Call `lib.quiver_database_read_time_series_files(handle, toCString(collection), ptr(outColumns), ptr(outPaths), ptr(outCount))`.
- Read count; if 0 return {}. Dereference both pointers.
- For each i: read column name via CString. Read path pointer; if falsy/zero -> null, else CString.
- Free with `lib.quiver_database_free_time_series_files(columnsPtr, pathsPtr, count)`.

**updateTimeSeriesFiles implementation:**
- Get entries from data. Build parallel string pointer tables for columns and paths (nullable: use `0n` for null path values).
- Call `lib.quiver_database_update_time_series_files(handle, toCString(collection), ptr(columnsPtrTable), ptr(pathsPtrTable), entries.length)`.
